6.图像标签
  
如
       <img src = "微信图片_20221204121516.jpg"  alt = "图片已丢失" title  = "她真漂亮"/> 

    src = "图像的路径，必须有"

    alt = "替换文本"

    title = "提示文本"

        height = "图片大小高度"  

    width = "图片大小宽度"

图像标签属性必须放到 标签名 img 的后面 

属性之间至少用一个空格隔开

属性不分先后

属性采用赋值对的方式  即：属性 = “属性值”

图片地址可以用相对路径或绝对路径 绝对路径\  相对路径/


7.超链接标签
<a></a>
链接语法格式 

    <a href = "链接地址" target = "打开链接方式"> 链接名 </a>

链接分为  

外部链接   href = https://www.*****

内部链接 即同一文件目录下的另一个 html 文档的名字  href = "gongsi.html"

空链接    href = "#"

下载链接 超链接的路径为 压缩包或者文件 点击链接就会下载 herf = "../images/微信图片_20221204121516.zip"

网页元素链接 在网页中的各种网页元素 如文本 图像 表格 音频 视频等都可以添加超链接

如  <a herf = "https://wallhaven.cc/"> <img src = "../images/wallhaven-m9dkgk.jpg"> </a> 给图片添加超链接

打开方式
 
_self 当前页面

_bland 新窗口

锚点链接  <a herf = "#锚点"></a>
锚点标签 <h1 id = "锚点"></h1>
